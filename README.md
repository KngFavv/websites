# MusicOnline

A full-stack music e-commerce web application developed using PHP, MySQL, HTML, CSS, and JavaScript. The platform allows users to browse music products, manage shopping carts and wishlists, place orders, and interact with a fully functional online store. An administrative dashboard is also included for managing products, users, and orders.

## Features

### Customer Features
- User registration and login
- Browse music products
- Product search functionality
- Product detail pages
- Shopping cart system
- Wishlist functionality
- Checkout process
- Order management
- Responsive interface

### Administrator Features

- Admin dashboard
- Product management
- User management
- Order management
- Store administration tools

## Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript
- jQuery

### Backend
- PHP

### Database
- MySQL

### Development Environment
- XAMPP
- Apache
- phpMyAdmin

## Installation

### Prerequisites
- XAMPP
- PHP
- MySQL

### Setup

1. Clone the repository:
git clone https://github.com/KngFavv/musiconline.git

3. Move the project folder into the XAMPP `htdocs` directory.

4. Start Apache and MySQL from the XAMPP Control Panel.

5. Open phpMyAdmin.

6. Create a new database.

7. Import the SQL file located in:
database/musiconline.sql

7. Configure database settings in:

connection.php

8. Open your browser and navigate to:
http://localhost/musiconline

# Recipe Website

A full-stack recipe sharing web application developed using PHP, MySQL, HTML, CSS, and JavaScript. The platform allows users to browse recipes, view recipe details, create accounts, contact administrators, and explore a collection of food recipes through a user-friendly interface. An administrative dashboard is included for managing recipes, users, and customer messages.

##  Features

### User Features
- User registration and login
- Browse recipe collection
- View recipe details
- Search and explore recipes
- Contact form for user enquiries
- Responsive website design
- Recipe category navigation
- Dynamic content loaded from a database

### Administrator Features
- Admin authentication
- Recipe management
- User management
- Customer message management
- Website content administration
