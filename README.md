# MusicOnline

A full-stack music e-commerce web application developed using PHP, MySQL, HTML, CSS, and JavaScript. The platform allows users to browse music products, manage shopping carts and wishlists, place orders, and interact with a fully functional online store. An administrative dashboard is also included for managing products, users, and orders.

## Features

### Customer Features
- User registration and login
- Browse music products
- Product search functionality
- Product detail pages
- Shopping cart system
- Wishlist functionality
- Checkout process
- Order management
- Responsive interface

### Administrator Features

- Admin dashboard
- Product management
- User management
- Order management
- Store administration tools

## Installation

### Prerequisites
- XAMPP
- PHP
- MySQL

### Setup

1. Clone the repository:
git clone https://github.com/KngFavv/musiconline.git

3. Move the project folder into the XAMPP `htdocs` directory.

4. Start Apache and MySQL from the XAMPP Control Panel.

5. Open phpMyAdmin.

6. Create a new database.

7. Import the SQL file located in:
database/musiconline.sql

7. Configure database settings in:

connection.php

8. Open your browser and navigate to:
http://localhost/musiconline
